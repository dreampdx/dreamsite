This is a 2D cellular automata processor with the following parameters:

  • LU: Live Underpopulation
  • LO: Live Overpopulation
  • DU: Dead Underpopulation
  • DO: Dead Overpopulation
  • AE: Area of Effect
  • NS: Number of Steps

Such that the following state transitions hold

  • T(L: L, D) = L if LU <= L <= LO, else T(L: L, D) = D
  • T(D: L, D) = L if DU <= L <= DO, else T(L: L, D) = D

Area of Effect increases radius of Moore neighborhood, scaling the other parameters as follows:

{LU, LO, DU, DO} = Floor({LU, LO, DU, DO} * ((3 + (AE - 1) * 2) ^ 2 - 1) / 8)

Thus, when AE is set to 1:

{LU, LO, DU, DO} = Floor({LU, LO, DU, DO} * 8 / 8)

{LU, LO, DU, DO} = {LU, LO, DU, DO}

And, when AE is set to 2:

{LU, LO, DU, DO} = 4 * {LU, LO, DU, DO}

So on.

I've spent some time drawing up classifications for the different emergent lifeforms from these parameters. These classifications are:

  • Blob – Lifeforms that grow continuously outwards.
  • Viable – Lifeforms that are unpredictable in whether they will grow or shrink
  • Non-Viable – Lifeforms that shrink continuously
  • Super-Viable – Lifeforms that fill all spaces in grid

I further parameterized Blobs as follows:

  • Border Quality:
    • Solid – Border of lifeform advances rigidly and as a "unified front"
    • Liquid – Border of lifeform advances in "waves" or "spills"
    • Gaseous – Border of lifeform advances in "pulses" or "particles"
  • Interior Quality:
    • Static – Lifeform finds arrangement in which cells do not transition
    • Pulsing – Lifeform finds arrangement in which some cells do not transition
    • Dynamic – Lifeform finds arrangement in which no cells do not transition

I further parameterized viables as follows:

  • Border Quality:
    • Closed – Lifeform tends towards closed shapes delineating life from empty space
    • Open – Lifeform tends towrads no delineation between life and empty space
  • Interior Quality:
    • Static – Lifeform finds arrangement in which cells do not transition
    • Pulsing – Lifeform finds arrangement in which some cells do not transition
    • Dynamic – Lifeform finds arrangement in which no cells do not transition

So, some classifications are

  • {0, 0, 1, 1, 1} = BGD
  • {2, 5, 1, 1, 1} = BLP
  • {2, 3, 3, 3, 1} = VOD
  • {0, 8, 1, 8, 1} = SV
