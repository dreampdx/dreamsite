Homotopy theory essentially began when Henri Poincare defined the fundamental group of a pointed topological space. Historically, this also marks one of the first functors to be described. Not only does a pointed space induce a group of loops (up to homotopy; this is crucial to get a group), but every basepoint-preserving continuous function induces a homomorphism of fundamental groups.

Continuous topological activity is transmuted into algebraic activity. Since every space has a continuous identity function mapping each point to itself, we can think of spaces as "static topological activity", and think of continuous functions in general as "topological activity". In fact, this characterizes category theory more generally. There is a definition of categories that doesn't use objects at all, only arrows. A category is a collection of arrows with two idempotent closure operators on it sending arrows to the identity arrows at their source and target, and composition is only defined where the images of these operators overlap (fg only exists if domain(f)=codomain(g), and then domain(fg)=domain(g) and codomain(fg)=codomain(f)). You can always recover objects from the arrow-only presentation, but technically they are just a convenient shorthand for static processes, where a "process" is just an arrow.

Homotopy theory in a category can begin as soon as you are able to identify some object, called an interval and usually denoted I, as the "worldline" of the identity arrow on a point. A line interval exhbits a point in stasis for a nonzero extent of time. Your category has to have enough products, because now we want to be able to multiply spaces, objects, et cetera with the interval. Since an interval has to have two endpoints, we can use the inclusion maps of the endpoints to get inclusion maps of any space X into the "X-shaped cylinder" X x I. This cylinder becomes the standard shape of homotopies between arrows that go from X to any other space Y. Since the interval is the point-shaped cylinder, a homotopy between points is just a path. But a homotopy between two paths, even ones that have different start and end points in the same ambient space Y, is a square in Y whose left edge is given by one path and whose right edge is given by the other path. The square then depicts a possible means of sliding one path continuously into the other.
