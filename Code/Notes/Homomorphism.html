The following are notes taken from the wikipedia entry on Homomorphisms

A Homomorphism is a structure-preserving map between two algebraic structures of the same type, meaning that some map between the two structures maintains some set of operations internal to the structures. The definition of this behavior is as such: a map f: A -> B preserves an operation u of arity k if f(u_A(a_1, ..., a_k)) = u_b(f(a_1), ..., f(a_k)), for all elements a_1, ..., a_k in A. To dissect this: there is an operation u in A. This operation, when applied to k elements in A, outputs some other element in A. This operation also exists in B. Some morphism f is considered structure-preserving (and thus is a homomorphism) if u is valid after f is applied to all the elements in u.

Rougly, I will begin to detail other special homomorphisms: Isomorphisms are essentially bijective homomorphisms, such that f o g = ID_B, and g o f = ID_A, with ID_A(x) = x in A, and ID_B(y) = y in B. Basically it means that a homomorphism is bidirectional, or reversible. Endomorphisms are homomorphims whose domain equals the codomain.
